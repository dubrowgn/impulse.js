# Impulse.js
Impulse.js is a high performance HTML5 game engine for JavaScript. It aims to be extremely flexible, without being overly complex.

Drive your imagination.

[![Build Status](https://app.travis-ci.com/dubrowgn/impulse.js.svg?branch=master)](https://app.travis-ci.com/dubrowgn/impulse.js)

## Use

```npm install @dubrowgn/impulse.js```

## Development

### Setup

```npm install --only dev```

### Building

Un-minified:

```npm run build```

 Minified: 
 
 ```npm run build-release```

### Testing

```npm run test```
